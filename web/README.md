# Synchatica Landing (`web/`)

MVP-лендинг Synchatica на Next.js 14 + TypeScript + Tailwind с motion/gsap/r3f.

## Команды

```bash
npm install
npm run dev
npm run lint
npm run build
npm run test:e2e
```

## Проверка

1. Откройте `http://localhost:3000`.
2. Проверьте Hero, 6 секций, CTA, страницы `/privacy` и `/terms`.
3. Для smoke-теста выполните `npm run test:e2e`.
